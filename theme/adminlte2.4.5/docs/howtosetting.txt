[G5.3] 반응형 테마 (AdminLTE 2.4.5)



mason님께서 만드신 테마(반응형 Bootstrap Admin Panel - AdminLTE 0.1.0 ( https://sir.kr/g5_theme/1383?page=3 ))"을보고
AdminLTE 사이트(https://adminlte.io/)의 최신 소스(AdminLTE 2.4.5: 제작 당시 최신버전)를 받아 다시 수정보았습니다.

부족한 점이 많으나 필요하신 분이 있을 것 같아 공유합니다.


[적용방법]
1. adminlte.zip 압축해제
2. 압축 해제된 adminlte 파일을 /theme에 업로드
3. 최고 관리자로 로그인
4. 관리 모드에서 기본환경 설정 변경: 모든 스킨을 "(테마) basic"으로 변경
  최근게시물, 검색, 접속자, FAQ, 모바일 최신게시물, 모바일 검색, 모바일 접속, 모바일 FAQ, 회원, 모바일 회원 스킨의 basic --> (테마) basic

5. notice라는 이름의 게시판 생성

  ※ 화면 상단의 쪽지 아이콘 옆에 공지사항 리스트 보기 아이콘때문에 필요
6. 관리자 모드에서 테마 설정 변경: adminlte 테마 선택
